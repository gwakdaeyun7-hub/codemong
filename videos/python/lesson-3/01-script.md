# 3강 — 변수와 자료형 (변수 / 숫자·문자열·불린 / `print()`)

- **목표 학습 결과**: 변수가 무엇인지 한 문장으로 설명하고, `이름 = 값` 한 줄에서 변수 이름과 값을 구분하며, 숫자·문자열·불린 세 자료형을 보고 분류한 뒤, `print()` 로 변수 값을 출력하는 4~6줄짜리 예제를 끝까지 읽어낼 수 있다.
- **prerequisite**: 1강 "파이썬 개요 & 개발환경" + 2강 "코딩의 표현 방법" 학습 완료
- **목표 분량**: 약 198초 (180초 ±20% 범위 내)
- **타깃 시청자**: 파이썬 1·2강을 본 입문자. **이번 강의에서 처음으로 실제 파이썬 코드를 만남**.
- **orientation**: landscape (1920x1080), 30fps
- **scene 수**: 13개
- **시나리오**: 자기소개 카드 (이름·나이·학생 여부 → 문자열·숫자·불린 균형 등장)

- **sceneId**: scene-06
- **duration_sec**: 16
- **narration**: |

    둘, 불린은 첫 글자가 반드시 대문자. 소문자 트루는 파이썬이 알아듣지 못합니다.

    - 화면 좌측 절반: 비교 컷
    - 좌측 위 박스: `5` (라벨 "숫자")

## Scene 10 — `print()` 로 출력하기

- **sceneId**: scene-10
- **duration_sec**: 22
- **narration**: |
    이제 화면에 보여줄 차례. 프린트 괄호 안에 변수 이름을 넣으면, 그 변수에 담긴 값이 화면에 출력됩니다.
    프린트 괄호 name 은 화면에 코드몽, 프린트 괄호 age 는 화면에 5, 프린트 괄호 is_student 는 화면에 트루.

- **on_screen**:

    - 결과 2: `5`
    - 결과 3: `True`
    - 각 코드 줄과 결과 줄을 잇는 회색 점선 화살표 (0.3초씩 fade-in)
- **note**: 학습 목표 4·5번 핵심. `print()` 의 의미를 narration 에서 풀어 설명. "프린트 괄호" 는 발음 사전 적용 (`print()` → "프린트 괄호"). 코드와 결과를 좌우 분리해서 학습자가 한눈에 매칭 가능. 결과의 `True` 가 따옴표 없이 그대로 출력되는 점은 narration 에서 짚지 않음 (4강 이후 깊이 다룸).

### 발음 사전 적용 확인

본 강의는 첫 실제 파이썬 코드 등장 강의로 영어 토큰이 다수. narration 안 영어 토큰 검수:

- `Python` → "파이썬" ✓ (scene 1, 4, 6 narration — 사전 기존)
- `print()` → "프린트 괄호" ✓ (scene 10 narration — **사전 신규 추가**)
- `True` → "트루" ✓ (scene 6, 9 narration — **사전 신규 추가**)
- `name` / `age` / `is_student` → 변수 이름은 영문 그대로 narration 에 노출 (scene 9, 10, 11). 입문자에게 처음 등장하는 변수 이름은 영문 그대로 듣고 익히는 게 자연스러움. TTS 가 "name" 을 "네임" 으로 읽고 "age" 를 "에이지" 로 읽는 게 표준 발음. video-voiceover-audio 단계에서 1차 검수 후 어색하면 한국어 음역 처리.

**한국어 토큰이지만 TTS 가독성 검수가 필요한 항목**:
- 코드 안 숫자 — narration 에서는 한자식 ("오", "육") 으로 명시 (scene 10, 11). 화면 텍스트는 아라비아 숫자 (`5`, `6`) 로 노출.
- "불린" — 한국어 외래어. TTS 표준 읽기로 충분.
- "자료형" — 한국어 합성어. TTS 표준.

**사전 신규 추가 단어 (이번 라운드)**:
- `print()` → "프린트 괄호"
- `True` → "트루"
- `False` → "폴스" (이번 영상엔 narration 미등장이지만 자료형 카드에 화면 노출되며, 다음 강의들에서 narration 등장 가능성 높아 선제 등록)
- `int` → "인트", `float` → "플롯", `str` → "스트링", `bool` → "불" — 본 영상에는 등장하지 않으나 자료형 시리즈로 묶어 선제 등록 (4강 이후 활용 예정)

**사전 미등록 처리한 단어**:
- `name`, `age`, `is_student` — 변수 이름은 매 영상마다 달라지므로 사전 등재가 부적절. narration 텍스트에 영문 그대로 두고 TTS 표준 발음에 맡김.
- `=` (등호) — narration 에서는 "등호" 한국어로 풀어 씀 (scene 4). 사전 매핑 불필요.
